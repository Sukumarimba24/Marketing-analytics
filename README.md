# Marketing-analytics

**About the dataset:**
This dataset contains details from bank's direct marketing campaigns (telephone calls). The main goal of these calls was to get customers to sign up for a "term deposit" (a type of savings account). It tracks over 41,000 interactions, recording customer details, the timing of the calls, and whether the customer eventually said "yes" or "no."

**Coclusion:**
The dataset shows a low-to-moderate conversion rate, typical for cold-calling marketing.
